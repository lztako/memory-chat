## Recommended Path สำหรับ Developer ใหม่

1. **First API call** → quickstart (`get-started.md`)
2. **Messages API** → request/response structure, multi-turn, system prompts, stop reasons
3. **Choose model** → compare capability + cost
4. **Explore features** → extended thinking, web search, file handling, structured outputs

## Developer Tools

- **Developer Console** — Prototype + test prompts ใน browser (Workbench + prompt generator)
- **API Reference** — Full Claude API + client SDK docs
- **Claude Cookbook** — Jupyter notebooks (PDFs, embeddings, ฯลฯ)

## Key Capabilities

- **Text & Code generation** — summarize, Q&A, extract data, translate, explain/generate code
- **Vision** — process/analyze images, generate text+code from images
